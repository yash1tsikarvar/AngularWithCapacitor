Tenant Evaluation is a
tenant screening service

Tenant Evaluation takes rental/purchase applications and process them for the property associations. This information helps the Associations make an informed desicion.
If the community requires a Credit Report, a Soft/Hard Equifax FICO Score 8 Algorithm or a TransUnion FICO 4 Algorithm will be processed.
Tenant Evaluation does not decide if you are approved. The Association does after reviewing your application.
